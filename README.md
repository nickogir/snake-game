# NEON ARCADE Snake

Browser-based Snake game (WebGL 3D render) hosted on GitHub Pages.

## Live URLs
- Game: https://nickogir.github.io/snake-game/
- Root hub: https://nickogir.github.io/

## Key Features
- Arrow/WASD movement
- SPACE to start/restart
- Wrap-around borders
- Score and best score tracking
- Keyboard-first fast replay loop
